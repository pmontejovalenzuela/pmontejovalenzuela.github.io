Replication of Card and Krueger (1994)

Minimum Wages and Employment: A Case Study of the Fast-Food Industry in New Jersey and Pennsylvania

Author

Paola Montejo Valenzuela

Published

Invalid Date

Introduction

Card and Krueger (1994) study what happened when New Jersey raised its minimum wage from $4.25 to $5.05 on April 1, 1992, while neighboring Pennsylvania kept the federal minimum wage at $4.25. The paper uses survey data from 410 fast-food restaurants in New Jersey and eastern Pennsylvania, collected just before and several months after the policy change.

The main question is whether the minimum wage increase reduced employment. In the standard competitive model, a binding wage floor should reduce demand for low-wage labor. Card and Krueger test that prediction with a difference-in-differences design: compare the before-after change in employment in New Jersey to the before-after change in employment in Pennsylvania.

This replication does three things:

  1. It reproduces part of Table 2.
  2. It reproduces the 9 numbers in Table 3, columns (i)-(iii), rows 1-3.
  3. It reproduces Table 4, column (i) and adds one extra result: Table 3, row 4 plus a graph of the main difference-in-differences pattern.

Load and clean the data

The original public.dat file is a fixed-width data file, so it has to be read using the column locations in the codebook. That detail matters for matching the published results closely.

Replicating Table 3: columns (i)-(iii), rows 1-3

The assignment asks for the 9 numbers in Table 3, columns (i)-(iii) and rows 1-3. These are:

  1. FTE employment before the policy, all available observations
  2. FTE employment after the policy, all available observations
  3. Change in mean FTE employment

A subtlety matters here. The paper notes that employment at the six permanently closed stores is set to zero, while employment at the four temporarily closed stores is treated as missing. For row 3, the authors compute the change in means as row 2 minus row 1, rather than taking store-level differences on the balanced sample.

row1_pa = njmin.loc[njmin["state"] == 0, "emptot"].mean()
row1_nj = njmin.loc[njmin["state"] == 1, "emptot"].mean()

row2_pa = njmin.loc[njmin["state"] == 0, "emptot2_closed0"].mean()
row2_nj = njmin.loc[njmin["state"] == 1, "emptot2_closed0"].mean()

row3_pa = row2_pa - row1_pa
row3_nj = row2_nj - row1_nj
row3_did = row3_nj - row3_pa

table3_rep = pd.DataFrame({
    "row": [

        "3. Change in mean FTE employment"

Replicating Table 4, column (i)

Table 4 turns the same idea into a regression. The dependent variable is the change in FTE employment. The sample is restricted to the 357 stores with available employment and starting wage data in both waves, except that permanently closed stores remain in the sample with wave-2 employment set to zero.

The SAS code distributed with the data uses the following sample rule:

  • keep observations with nonmissing change in employment, and
  • keep stores that are permanently closed or have nonmissing wage change data.
reg_sample = njmin.loc[njmin["demp_all"].notna() & (njmin["closed"] | njmin["dwage"].notna())].copy()

X = sm.add_constant(reg_sample["nj"])
model1 = sm.OLS(reg_sample["demp_all"], X).fit()

coef_nj = model1.params["nj"]
se_nj = model1.bse["nj"]

pd.DataFrame({
    "term": ["New Jersey dummy"],
    "estimate": [coef_nj],
    "std_error": [se_nj],
    "n": [len(reg_sample)]
}).round(3)
term estimate std_error n

This reproduces Table 4, column (i) very closely:

  • New Jersey dummy = 2.326
  • Standard error = 1.192
  • N = 357

The published table reports 2.33 with standard error 1.19, so the replication matches essentially exactly.

Something additional

For the extra step, I replicate Table 3, row 4, which uses the balanced sample only: stores with available employment data in both waves. This is a useful comparison because it shows the main finding is not driven only by the changing composition of stores with observed employment.

balanced = njmin.loc[njmin["emptot"].notna() & njmin["emptot2"].notna()].copy()
balanced["demp_balanced"] = balanced["emptot2_closed0"] - balanced["emptot"]

row4_pa = balanced.loc[balanced["state"] == 0, "demp_balanced"].mean()
row4_nj = balanced.loc[balanced["state"] == 1, "demp_balanced"].mean()
row4_did = row4_nj - row4_pa

A graph of the main difference-in-differences result

The paper mainly presents these values in a table, but the basic identification strategy is easier to see in a graph.

plot_data = pd.DataFrame({
    "state": ["Pennsylvania", "Pennsylvania", "New Jersey", "New Jersey"],
    "period": ["Before", "After", "Before", "After"],
    "mean_fte": [row1_pa, row2_pa, row1_nj, row2_nj]
})

for state, g in plot_data.groupby("state"):
    plt.plot(g["period"], g["mean_fte"], marker="o", label=state)

plt.title("Mean FTE Employment Before and After the NJ Minimum Wage Increase")
plt.xlabel("")
plt.ylabel("Mean FTE employment")
plt.legend(title="")
plt.tight_layout()
plt.show()

The graph makes the difference-in-differences logic clear. Pennsylvania stores lost employment between the two surveys, while New Jersey stores were roughly flat to slightly higher. The resulting gap is positive, not negative, which is inconsistent with the simple prediction that a higher minimum wage must reduce employment.

Why difference-in-differences helps here

Difference-in-differences helps the authors draw a causal conclusion because it removes time-invariant differences between New Jersey and Pennsylvania stores and nets out any common regional shock that affected both states between the two interviews. Under the parallel-trends assumption, the remaining difference in employment changes can be interpreted as the effect of the New Jersey minimum wage increase.

In this setting, Pennsylvania is useful as a comparison group because it is geographically close and economically similar, but it did not change its minimum wage during the same period. That makes it a plausible counterfactual for what would have happened to New Jersey fast-food employment in the absence of the policy.

Conclusion

This replication successfully reproduces the main result of Card and Krueger (1994).

  • The descriptive statistics from Table 2 match closely.
  • The key difference-in-differences result from Table 3 is about 2.76 FTE workers.
  • The regression estimate in Table 4, column (i), is 2.33 with a standard error of 1.19.

Overall, the replication confirms the paper’s central finding: in this case, the New Jersey minimum wage increase did not reduce employment in fast-food restaurants relative to Pennsylvania. If anything, employment rose slightly in New Jersey relative to the comparison group.
